# Setup for Friends

1. Install Hammerspoon from https://www.hammerspoon.org/
2. Unzip `ClearNotificationsInstaller.zip`.
3. Double-click `ClearNotificationsInstaller.command`.
4. Open System Settings when prompted.
5. Enable Hammerspoon under Privacy & Security > Accessibility.
6. Double-tap Command to clear visible notifications.

If it does not work, quit and reopen Hammerspoon, then choose Reload Config from the Hammerspoon menu bar icon.
